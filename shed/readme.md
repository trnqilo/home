# Rusty Shed
## functional examples

### rx/rayon

```bash
cd reactive_shed && cargo run
```

### tokio threading with glium/opengl

```bash
cd concurrency_shed && cargo run
```

### yew.rs basic project

```bash
cd yew_shed && trunk serve
```

### vendor shed

```bash
cd vendor_shed && cargo run
```

### test shed

```bash
cd test_shed && cargo test
```
